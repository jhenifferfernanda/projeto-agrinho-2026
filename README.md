# Projeto Agrinho

## Descrição

Este projeto foi desenvolvido para o Programa Agrinho com o objetivo de promover a conscientização sobre sustentabilidade, educação e a importância da integração entre o campo e a cidade. Através de recursos digitais, o projeto busca apresentar informações de forma clara, educativa e acessível para todos os públicos.

## Objetivos

- Incentivar a educação ambiental;
- Valorizar a conexão entre campo e cidade;
- Promover práticas sustentáveis;
- Utilizar a tecnologia como ferramenta de aprendizado.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Autor

Projeto desenvolvido para participação no Programa Agrinho.
